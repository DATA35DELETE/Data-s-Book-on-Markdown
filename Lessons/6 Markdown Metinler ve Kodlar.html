Metinler ve Kodlar

Metinler

Metin yazabilmek için herhangi bir işarete gerek yoktur. Direk yazılabilir. Ayrıca kalın, italik ve vurgulu kelimeler yazılabilir. "<br>" ile satır atlanır. Aksi taktirde metinler tek bir paragraf şeklinde gözükür.

Kullanımı

# Metinler

*Bu bir italik metin* <br>
**Bu bir kalın metin** <br>
Bu metin `vurgu` içeriyor

Kod Yazma

Kod yazmak için Markdown da ters tırnak işaretleri kullanılır. Üç artı üç toplamda altı tane konulur. Kod yazılan kısımın hemen üstünde ise, kodların hangi dile ait olduğunu ifade eder. İfade edilen dile görede kodlar renklendirilir.

Kullanımı

# Kodlar

Bu bir Java kodu
```java
public class Main
{
    public static void main(Stirng[] args)
    {
        System.out.println("Hello World");
    }
}
```

Yukarıdaki kodun aynı işi yapan bir Rust kodu
```rust
fn main()
{
    println!("Hello World");
}
```
