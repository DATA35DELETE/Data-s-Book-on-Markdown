Markdown Nedir?

Markdown, John Gruber 2004 yılında Aaron Swartz ile geliştirilen bir işaretleme dilidir.
Markdown genellikle yazılım projelerinde, kullanıcıya veya geliştirciye projenin nasıl kullanılcağını ve yapısını yazmak için kullanılır. Markdown ile geliştiriciler için bütün kodları okumak yerine "md" uzantılı dosyadaki içeriğe okuyarak projelerini geliştirmesini sağlar.

Markdown'nun Özellikleri

  • Başlıklar
  • Metinler
  • Metin Stilleri
  • Maddeler
  • Renkli Kod Sunumları
  • Tablolar
  • Resimler
  • Linkler

Markdown Dosyası Yazma

Markdown dosyası yazmadan önce, Visual Studio Code'u indirilim ki Code, hem Markdown yazmak hem de görüntülemek için ortam sunmaktadır.

Visual Studio Code Kurulumu

Visual Studio Code'u kurabilmek için, sitesine gidip, kurucuyu indirelim.
İnen kurucuyu çalıştıralım ve gerekli işlemleri yaptıktan sonra Visual Studio Code'muzu açalım. Ardından bir klasör seçelim ve sonu ".md" ile biten bir dosya oluşturalım. Sağ üsteki çıkan ikona basalım ve artık Markdown yazabiliriz.

Kod Örneği

İlk olarak bir Markdown örneği verelim
# Hello

- Hello
- Merhaba
- Hola

Bunların Türkçesi **Merhaba**'dır
